# AtEast ATM System

A Java ATM Management System that simulates banking operations through a console application. This project demonstrates object-oriented programming, file handling, modular architecture, and transaction management.

## Features:

### 📍User Authentication
- ✅Secure Login
- ✅Account Registration
- ✅PIN Verification

### 📍Banking Operations
- ✅Deposit Money
- ✅Withdraw Money
- ✅Transfer Funds
- ✅Balance Inquiry

### 📍Transaction History
- ✅Stores every transaction
- ✅Displays complete transaction logs
- ✅Timestamp for every operation

### 📍File Management
- ✅Persistent account storage
- ✅Automatic data loading
- ✅Automatic data saving

### 📍Modular Architecture
- ✅Separate packages
- ✅ Clean code organization
- ✅ Easy maintenance

## Technologies Used:

- 🚩Java
- 🚩Object-Oriented Programming (OOP)
- 🚩File Handling
- 🚩Collections Framework (HashMap, ArrayList)
- 🚩Date & Time API

## How to Get Started⁉️

### ☑️Clone the Repository

```bash
git clone https://github.com/Frenzy-FC/AtEastATM.git
```

### ☑️Navigate to the Project

```bash
cd AtEastATM
```

### ☑️Compile

```bash
javac -d out src/app/Main.java src/utils/*.java src/data/*.java src/model/*.java src/file/*.java src/service/*.java
```

### ☑️Run

```bash
java -cp out app.Main
```
